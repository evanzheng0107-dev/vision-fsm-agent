# Visual Grid World Demo

A fully synthetic, headless, reproducible demo environment for Vision FSM Agent.

## Quick Run

```bash
# From the project root
python examples/visual_grid_world/run_demo.py --steps 35
```

Or using the launcher:

```bash
python run.py --start --steps 35
```

## What It Does

The demo creates a 12×12 grid world with:
- **Goals** (green crosshairs) — navigation targets
- **Items** (gold circles) — collectibles
- **Buttons** (blue squares) — interactable elements

An agent (red diamond) navigates the grid, collecting items and pressing
buttons — all driven by real template matching and FSM transitions.

## Expected Output

```
[demo] World: 12x12, goals=[(3, 2), (6, 5)], items=3, buttons=1
  step   1 | fsm=IDLE  pos=(1, 1) | items[...] buttons[.]
  ...
  step  19 | fsm=IDLE  pos=(3,10) | items[III] buttons[B]
[demo] All items collected and buttons pressed. Done!
```

## Configuration

Edit `examples/visual_grid_world/config.yaml` to change:
- Grid size (`demo_grid_w`, `demo_grid_h`)
- Random seed (`demo_seed`)
- Confidence threshold (`confidence_default`)

## Assets

Template images in `examples/visual_grid_world/assets/` are programmatically
generated by `scripts/generate_demo_assets.py`.

## Save Frames

```bash
python examples/visual_grid_world/run_demo.py --save demo_frames --steps 20
```

Frames are saved as `demo_frames/frame_000.png`, `frame_001.png`, etc.
